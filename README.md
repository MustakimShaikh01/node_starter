# Node Express API Complete (15 CRUD endpoints)

This project includes:
- 5 resources (User, Post, Comment, Product, Order) each with 3 endpoints = 15 CRUD endpoints
- JWT auth (register/login) for protected endpoints
- Swagger UI at /api-docs
- Rate limiting (throttling) applied globally and per-route
- SQLite by default (easy local testing) and Postgres-ready config
- Seed data and Postman collection

Quick start:
1. Copy `.env.example` to `.env` and set JWT_SECRET.
2. `npm install`
3. `npm run seed`
4. `npm run dev`
5. Swagger: http://localhost:3000/api-docs
